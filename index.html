SAGA-Python

SAGA-Python is the infrastructure access layer of the RADICAL Cybertools stack. It provides a homogeneous programming interface to the majority of production HPC queuing systems, Grid- and Cloud-services. SAGA-Python supports XSDE and OSG resources as well as academic and commercial cloud computing platforms.

RADICAL-Pilot (BigJob)

RADICAL-Pilot (BigJob) is the infrastructure abstraction layer of the RADICAL Cybertools stack. It provides a programming library to create ad-hoc resource overlay and multilevel scheduling environments based on Pilot-Jobs. RADICAL-Pilot can marshal and O(10k) distributed cores and schedule O(10k) tasks in a single application context across multiple distributed distributed HPC resources.
